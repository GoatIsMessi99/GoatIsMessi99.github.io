1. lambda的特性与用法

1.1 lambda与普通函数的区别

首先我们来看下基本lambda函数的构成：

[ ]( ) -> void { };

再来看看普通函数的构成:

void func( ){ } 或 auto func( ) -> void{ }

通过对比，我们可以发现，lambda表达式与普通函数有如下异同：

相同点：

  • 函数形参列表
  • 函数体

不同点：

  • lambda多了 [ ] 部分，我们称之为捕获列表
  • lambda没有显示的函数名

1.2 lambda捕获列表

所谓捕获列表，可以理解为是函数形参列表的一部分，因为函数体内部是无法直接使用函数体外部的变量的，因此lambda表达式也成为闭包 , 通过捕获列表获得的变量，可以被函数体内部使用。

lambda捕获列表形式有以下5种：

  • 空捕获 [ ] ,
  • 值捕获 [param]
  • 引用捕获[&param]
  • 隐式捕获 [&] [=]
  • 表达式捕获 [ exp]

捕获列表与普通函数的形参列表不同的一点是：lambda表达式在创建的时候，就对函数参数进行了初始化，而不是调用函数时才初始化
